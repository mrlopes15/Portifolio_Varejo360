Varejo 360 – Análise de Ocorrências no Varejo

Projeto de análise de dados focado em ocorrências operacionais no varejo, utilizando Python, IA generativa, PostgreSQL(NEON) e Power BI.

O objetivo é identificar padrões de insatisfação do cliente, com foco especial em rupturas virtuais, e apoiar a tomada de decisão operacional.
